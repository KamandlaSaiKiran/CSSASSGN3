Absolute Positioning Relative Positioning

Syntax is position:absolute;

Syntax is position:relative;

With absolute positioning, an element is positioned relative to its nearest positioned ancestor

With relative positioning, an element can be moved up,down, left or right from its normal position

It will break the normal document flow to position the element on the page

It will not break the normal document flow to position the element on the page

The element will not leave any space in its normal document

The element will leave the space at its original position and other content will not be adjusted to fit into any gap left by the element
